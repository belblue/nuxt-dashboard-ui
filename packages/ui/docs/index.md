# Nuxt Dashboard UI

A collection of Vue components for building dashboard interfaces with Nuxt.

## Installation

```bash
pnpm add nuxt-dashboard-ui
```

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['nuxt-dashboard-ui']
})
```

## Components

- [StatCard](./components/StatCard.md) - Display key metrics with trend indicators

## Features

- SSR/Hydration safe
- Dark mode support
- Fully typed with TypeScript
- Customizable via CSS variables
- Slot-based flexibility
- i18n ready
